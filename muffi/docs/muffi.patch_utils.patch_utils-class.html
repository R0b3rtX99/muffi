poly_eax_dword(self, value=GENERIC_PATCH_VALUE, flavor=0)

source code 
This function will create a polymorphic set of instructions to write out the opcode bytes that will set EAX to a DWORD value. The flavor parameter is important, the smaller the flavor index the smaller the size of the patch.
Parameters:
  • value (DWORD) - (Optional) The value that the DWORD in EAX should be.
  • flavor (INTEGER) - (Optional) The flavor dictates what style of DWORD patch should be applied. Randomized if left as default value.
Returns:
Returns the instructions for poly-patching EAX.
Raises:
  • mfx - An exception is raised on failure.

poly_eax_zero(self, flavor=0)

source code 
This function creates a polymorphic set of instructions that zeroes out the EAX register.
Parameters:
  • flavor (INTEGER) - (Optional) The flavor dictates what style of patch is generated to zero out EAX.
Returns:
Returns the instructions for poly-patching an EAX = 0.
Raises:
  • mfx - An exception is raised on failure.
